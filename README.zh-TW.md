# ASI 競賽

![ASI 競賽遊戲截圖](docs/screenshots/gameplay.png)

語言：[EN](README.md) | [JA](README.ja.md) | [FR](README.fr.md) | [ZH-CN](README.zh-CN.md) | [ZH-TW](README.zh-TW.md)

ASI 競賽是一款在瀏覽器中執行的 3D 即時戰略遊戲。四家 AI 實驗室風格的陣營圍繞算力、資料、人才、政府影響力、公眾信任、風險控制和對齊展開競賽。最先完成 ASI 訓練的一方會結束比賽；摧毀所有對手總部也能獲勝。

> 這是一個非官方戲仿專案。專案與 OpenAI、Anthropic、Google DeepMind、xAI 或任何真實實驗室無關聯、未獲背書；陣營設定是基於公開印象的遊戲化改寫，不包含真實人物。

## 從原始碼執行

本倉庫保存的是從 Fable 生成包中抽出的共享瀏覽器遊戲原始碼，並已經包含 `vendor/three.module.js`，因此原始碼版可以直接透過任意本機靜態伺服器執行。

```bash
python3 -m http.server 8000
# 開啟 http://localhost:8000
```

執行無介面模擬測試：

```bash
npm test
# 等同於 node test/headless.mjs
```

## 玩法

- 算力來自總部、資料中心和可佔領的 GPU 叢集。
- 資料由研究員從地圖節點採集，高階實驗樓可自動產出合成資料。
- 影響力來自國會遊說，可用於出口管制、算力補貼、監管調查和公關攻勢。
- 人才決定單位上限，信任影響招募成本和被挖角風險。
- 搶跑研發會累積風險，對齊研究可降低事故壓力並影響結局。
- 開局可選兩種戰場時段：☀ 白天（烈日金色草原漂在暗夜虛空中，白色園區）與 🌆 黃昏（原版暮色）。
- 畫面走電影化的風格化寫實：全局調色（降飽和、暗角、膠片顆粒），實拍地表紋理加上每個建築場地的真實挖填方切坡，深色機加工金屬甲板場坪配陣營色四角燈柱取代道具式底座，建築外皮對照原參考影片重做——珍珠白合金板與條帶玻璃、帶環境反射，不再是混凝土原色——立面保留人體尺度的檢修細節（爬梯、落水管、進風格柵），建築語彙取材大量現實高科技園區並推向超越現實，人物為寫實比例的骨骼動畫模型。
- 動態天氣掠過戰場：雲影緩緩掃過地表，陣雨拉下雨幕、壓暗太陽，雨過金色陽光回歸；白天調色走明媚路線——暖陽關鍵光、橙青分離調色、高光微微閃耀。
- 底部 HUD 是一整套專業 RTS 指揮台：戰術地圖站位帶四角托架與掃描光帶，指揮資訊站位帶陣營銘牌、分段血條和生產佇列格，作戰指令台則是一排帶熱鍵鍵帽、費用條與鎖定/資源不足狀態的立體指令鍵——三塊削角裝甲面板鉚接在貫穿全螢幕的底盤導軌上，整體染上你所選陣營的塗裝色。
- 地形即玩法：四條外緣走廊被台地絕壁截斷，大軍只能從 GPU 集群所在的隘口通過；坡地減速，陡壁不可通行、不可施工；高地單位視野更遠、俯擊傷害 +30%，仰攻傷害 −20%。
- 世紀帝國式營運層：每種建築內都有一次性經濟科技可研究（光互連、資料管線、紅隊演練、過程監督……），總部另設帶滑點的算力現貨市場，應急調劑資源。
- 對手 AI 用同一套打法：按性格優先級買科技、上市場調劑、讀你的兵種構成練剋制兵、給遠礦派護衛、殘兵及時撤退，還會朝你上次進攻的方向補防火牆塔。
- 戰場之外還有一整層行業元經濟：各實驗室股價與增發融資、被加密牛市/產能短缺/開源衝擊推動的硬體指數、雲服務模式（出租算力換資料與影響力、順帶壓低全行業硬體價格），以及戲仿的明星研究員——會退圈、跳槽、被天價挖角，或在地圖上自立門戶創辦新勢力，誰都可以在其 IPO 之前收購。
- 終局是「湧現」：最終訓練分五段甦醒，每段性情由當下對齊值決定——遞迴自我優化讓時鐘融化、檢查點商用帶來收入並壓價對手的算力轉售、規模化敘事投放（或公開安全評測）扭轉信任、贏家通吃的人才虹吸（外加收買內線）抽乾對手、最後一手要麼防禦性披露、要麼合法卻狠辣的電網容量擠兌；兩條訓練同跑則進入競爭性超頻。
- 戰場覆蓋戰爭迷霧：單位與建築提供視野，探索過的區域轉為灰色記憶，敵方建築只保留你最後一次看到的樣子；小地圖同步受迷霧限制，唯有 ASI 訓練的光柱全圖可見。
- 勝利路線包括 Gen-2、Gen-3、Gen-4 / AGI 到 ASI 訓練，也可以摧毀所有對手總部。

可玩陣營受 OpenAI、Anthropic、Google DeepMind 和 xAI 啟發，每一方都有不同的經濟或安全傾向加成。

## 專案結構

```text
index.html            瀏覽器入口
css/                  HUD、開始頁、戰場覆蓋層樣式
js/sim/               確定性模擬層，不依賴 DOM 或 three.js
js/view/              three.js 表現層、地形、建築、角色、特效
js/ui/                HUD 與遊戲內教學
js/audio/             WebAudio 音效和環境配樂
js/shared/            模擬與渲染共用函式
test/headless.mjs     Node 模擬層測試
vendor/               vendored three.js v0.170.0
packaging/            從 macOS 和 Windows 包中抽出的啟動器原始碼
```

模擬層以固定 0.1 秒步長推進。渲染層讀取狀態並播放插值表現效果。AI 玩家與真人玩家使用同一套命令 API。

## 資產說明

原始提示詞要求使用真實下載素材。當前 Fable 產出的包實際只依賴下載版 Three.js；建築、角色幾何、骨骼動畫姿態、貼圖、音效和環境音樂均由專案程式碼程序化生成。在此基礎上，倉庫現已加入兩組來自 Poly Haven 的 CC0 實拍地表紋理（`assets/textures/`：航拍草地/岩石漫反射與法線貼圖、乾土漫反射），用於地形渲染；其餘素材仍為程序化生成。本倉庫保留這一實作狀態，並把 Three.js 固定為 `0.170.0`，以便原始碼版直接執行。
