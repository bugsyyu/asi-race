ASI 竞赛 — Windows 版使用说明
==============================

这是一款在浏览器里运行的即时战略游戏（俯视角，AoE 风格），
讲的是四家 AI 实验室冲刺超级智能的故事。纯属恶搞，与任何真实公司无关。
本包与 macOS 版是同一款游戏。

一、怎么开始
------------
解压后双击「启动游戏.cmd」。
会出现一个黑色控制台窗口 —— 那就是游戏的本地服务器，随后浏览器会自动
打开游戏页面。全程只使用 Windows 自带组件（PowerShell），不安装任何东西，
不需要管理员权限，也不会弹防火墙授权（服务器只监听本机 127.0.0.1）。

【第一次可能被 SmartScreen 拦下】提示"Windows 已保护你的电脑"时：
  点「更多信息」→「仍要运行」。
  也可以在解压 zip 之前：右键 zip 文件 → 属性 → 勾选「解除锁定」→ 确定，
  再解压，就不会出现该提示。

【引擎已内置，无需联网】本包已内置游戏引擎 three.js（app\game\vendor\），
解压即可完全离线游玩。若该文件意外丢失，启动器会在联网启动时自动补齐；
也可在任何能联网的设备手动下载
    https://cdn.jsdelivr.net/npm/three@0.170.0/build/three.module.js
放回 app\game\vendor\ 目录（保持文件名不变）。

二、怎么退出
------------
关闭那个黑色控制台窗口即可（或在窗口里按 Ctrl+C）。
只关浏览器标签页不会停止服务器。

三、常见问题
------------
· 浏览器建议使用 Edge / Chrome / Firefox 的近两年版本（系统自带的 Edge 即可）。
· 公司/学校电脑若通过策略禁用了 PowerShell，本启动器将无法工作。
· 个别杀毒软件可能对"脚本起本地服务器"行为提示 —— 脚本就在
  app\launcher.ps1，纯文本可自行审阅，选择放行即可。
· 端口 8423–8467 被占满时，可先设置环境变量 ASI_PORT 再运行。

四、出了问题怎么办
------------------
· 游戏页面若出现红色报错浮层，把里面的文字原样复制发回来。
· 启动器日志在 %TEMP%\asi-race-launch.log（控制台窗口里也有同样输出），
  一并发来更好。

五、给爱折腾的人
----------------
完整源代码在 app\game\ 目录，其中 README.md 有架构说明与开发指南；
模拟层自带测试：node test\headless.mjs（需自装 Node，仅开发用）。
